# EOS API - Event Operating System

O **EOS API** é o núcleo de processamento do sistema de gestão de eventos. Desenvolvido com uma arquitetura em Java, ele gerencia a lógica de negócio, persistência de dados e integração entre sócios e eventos.

## 🚀 Tecnologias
* **Java 21**
* **Spring Boot 3**
* **Spring Data JPA**
* **PostgreSQL** (Banco de dados relacional)
* **Maven** (Gerenciador de dependências)
